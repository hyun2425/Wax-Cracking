# 온라인 왁뿌볼 (Online Wax-Cracking Ball)

## 1. 프로젝트명

온라인 왁뿌볼 (Online Wax-Cracking Ball)

## 2. 프로젝트 소개

온라인 왁뿌볼은 최근 유튜브와 틱톡 등에서 인기를 끌고 있는 왁스볼 깨기(왁뿌볼) 콘텐츠를 웹으로 구현한 ASMR 시뮬레이션 서비스이다.

실제 왁스볼은 한 번 깨면 다시 사용할 수 없지만, 온라인 왁뿌볼은 다양한 종류의 왁스볼을 선택하여 반복적으로 깨뜨릴 수 있다.

사용자는 원하는 왁스볼을 선택한 후 냉동실에 넣어 굳히고, 마우스로 직접 눌러 깨뜨릴 수 있다. 냉동 시간에 따라 왁스볼의 강도와 파괴음이 달라지며, 실제 왁스볼을 깨는 듯한 만족감을 제공하는 것이 목표이다.

## 3. 개발 목적

- 유행하는 왁스볼 깨기 콘텐츠를 온라인으로 구현한다.
- ASMR 요소를 활용하여 사용자에게 재미와 만족감을 제공한다.
- 냉동 시간에 따른 변화를 통해 기다리는 재미를 추가한다.
- 실제 왁스볼과 달리 반복적으로 즐길 수 있는 콘텐츠를 제공한다.

## 4. 주요 기능

### 4-1. 왁뿌볼 선택

사용자는 원하는 종류의 왁뿌볼을 선택할 수 있다.

#### 두바이 왁뿌볼

- 초콜릿 색상 기반 디자인
- 금색 포인트 효과
- 묵직한 파괴음 제공

#### 솜사탕(삼색) 왁뿌볼

- 분홍, 하늘, 보라색 조합
- 부드러운 이미지
- 경쾌한 파괴음 제공

#### 청사과 왁뿌볼

- 연두색 디자인
- 상쾌한 이미지
- 시원한 파괴음 제공

### 4-2. 냉동실 시스템

선택한 왁뿌볼을 냉동실에 넣어 굳힐 수 있다.

사용자의 대기 시간을 줄이기 위해 게임 시간 시스템을 적용한다.

- 1초 = 1분
- 최대 냉동 시간: 20분
- 실제 대기 시간: 최대 20초

| 냉동 시간 | 상태 |
| --- | --- |
| 0~5분 | 말랑 |
| 5~10분 | 보통 |
| 10~15분 | 단단 |
| 15~20분 | 매우 단단 |
| 20분 | 최고 경도 |

### 4-3. ASMR 사운드 시스템

냉동 시간에 따라 서로 다른 파괴음을 제공한다.

| 냉동 시간 | 소리 |
| --- | --- |
| 0~5분 | 뽁 |
| 5~10분 | 뽁각 |
| 10~15분 | 빠각 |
| 15~20분 | 와작 |
| 20분 | 와그작!!! |

또한 왁뿌볼 종류에 따라 음색이 다르게 적용된다.

### 4-4. 마우스 클릭 파괴 시스템

사용자는 마우스로 왁뿌볼을 직접 눌러 깨뜨릴 수 있다.

파괴 과정:

1. 클릭 시작
2. 균열 발생
3. 균열 확대
4. 완전 파괴
5. 파괴음 재생

냉동 시간이 길수록 균열 효과가 더욱 화려하게 나타난다.

### 4-5. 균열 애니메이션

파괴 진행률에 따라 왁뿌볼에 금이 가는 효과를 제공한다.

예시:

- 25%: 작은 균열
- 50%: 중간 균열
- 75%: 큰 균열
- 100%: 완전 파괴

## 5. 기대 효과

- 실제 왁스볼 깨기의 재미를 온라인에서도 경험할 수 있다.
- 반복 플레이가 가능하여 지속적인 이용이 가능하다.
- 다양한 종류의 왁뿌볼과 소리를 통해 사용자 만족도를 높일 수 있다.
- ASMR 콘텐츠를 좋아하는 사용자들에게 새로운 즐길 거리를 제공할 수 있다.
